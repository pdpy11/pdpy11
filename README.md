# PDPy11

PDPy11 is a cross-platform assembler for PDP-11 computers.


## Requirements

- Python 3.6 or later
- No Python modules required
- Any platform (Windows, Mac OS, Linux)


## Installation

Use pip.

```shell
$ pip3 install pdpy11
```

(some modern systems use `pip` instead of `pip3` for Python 3 packages)
